# Privacy Policy - AI Revenue OS

This Privacy Policy explains how **AI Revenue OS** (the "Application") collects, stores, and processes data when integrated with the Pinterest API.

## 1. Information Collected
AI Revenue OS accesses Pinterest account data solely for internal automation and analytics.
* **Pins and Boards Data**: The Application retrieves board details and lists pins created by the authenticated account.
* **Performance Metrics**: The Application retrieves organic performance statistics (such as impressions, outbound clicks, and saves) of pins published by the Application.
* **No Personal Data Harvesting**: The Application does not collect, harvest, or store personal user profiles, emails, or personal identification data of other Pinterest users.

## 2. Use of Data
All retrieved metrics and metadata are used strictly for:
* Calibrating internal prediction engines.
* Tracking and ranking creative performance under the Genome Library.
* Logging decision cycles inside the local Decision Ledger.

## 3. Data Storage and Security
* **Local Database Only**: All data retrieved from the Pinterest API is stored locally inside private SQLite databases on your machine.
* **No Third-Party Sharing**: The Application does not sell, lease, share, or transmit any account data or analytics to third-party services.

## 4. User Control and Data Deletion
Since all data is stored locally, you have full control over your data:
* You can delete all collected metrics and logs at any time by deleting the local SQLite database files (`knowledge/*.db`).
* You can request the deletion/archiving of any published Pin directly through the Application's archiving capabilities.

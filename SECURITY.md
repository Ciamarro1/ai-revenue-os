# Security Policy

## Supported Versions

Atualmente, apenas a versão listada abaixo recebe patches de segurança ativos.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Por favor, **NÃO** reporte vulnerabilidades de segurança via Issues públicas no GitHub. 

Para reportar uma vulnerabilidade ou risco estrutural de segurança (especialmente relacionada a prompt injection, vazamento de telemetria ou RCE na arquitetura do Orquestrador), envie um e-mail diretamente para a nossa equipe de engenharia em `security@airevenue.os`.

A equipe de segurança responderá ao relatório em até 48 horas. Se o relatório for validado, trabalharemos para emitir um patch o mais rápido possível e incluí-lo no próximo release menor de segurança.
